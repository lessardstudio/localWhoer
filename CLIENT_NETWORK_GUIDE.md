# Инструкция подключения к корпоративной VPN сети

## Что вы получите после подключения:

✅ Доступ к корпоративным сервисам
✅ Возможность общаться с другими сотрудниками в сети
✅ Доступ к файловому серверу
✅ Безопасное зашифрованное соединение

## 1. Установка OpenVPN клиента

**Windows:**
- Скачайте: `https://openvpn.net/community-downloads/`
- Установите OpenVPN GUI

**macOS:**
- Скачайте Tunnelblick: `https://tunnelblick.net/`

**Linux:**
```bash
sudo apt install openvpn
```

**Android/iOS:**
- Установите "OpenVPN Connect" из магазина приложений

## 2. Импорт конфигурации

Вы получили файл `ваше_имя.ovpn` от администратора.

**Windows:**
- Скопируйте `.ovpn` в `C:\Program Files\OpenVPN\config\`
- Запустите OpenVPN GUI
- Правой кнопкой на иконке в трее → Connect

**macOS/Linux:**
- Двойной клик на `.ovpn` файле

## 3. Проверка подключения

После подключения:
```bash
# Проверить IP адрес VPN
ip addr show tun0     # Linux/macOS
ipconfig /all         # Windows

# Вы должны увидеть: 10.8.0.X
```

## 4. Доступные сервисы

### whier-app (Веб-приложение)
```
http://172.20.0.10:3000
или
http://whier.local:3000
```

### Файловый сервер
```
Windows: \\172.20.0.20\shared
macOS:   smb://172.20.0.20/shared
Linux:   smb://172.20.0.20/shared
```

### Другие сотрудники
Узнайте IP адреса коллег (10.8.0.X) для прямого обмена файлами или общения.

## 5. Тестирование связи
```bash
# Проверить доступ к whier-app
curl http://172.20.0.10:3000

# Проверить связь с другими клиентами
ping 10.8.0.5  # IP коллеги
```
